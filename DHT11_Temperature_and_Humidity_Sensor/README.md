• DHT11 Temperature and Humidity Sensor

•• Description
This Arduino project uses the DHT11 sensor to measure both temperature and humidity, then displays the results on the Serial Monitor.

•• Components
- Arduino Uno
- DHT11 Sensor
- Jumper wires

•• How It Works
The DHT11 sensor reads temperature and humidity values.
The Arduino processes the data and shows it on the Serial Monitor.

•• Output
- Temperature in °C
- Humidity in %
