• Smart Gas Detection and Alert System

•• Description
This Arduino project detects gas using a gas sensor and activates a buzzer when the gas level exceeds a predefined threshold.

•• Components
- Arduino Uno
- MQ-2 Gas Sensor
- Buzzer
- Jumper wires

•• How It Works
The gas sensor continuously reads the gas level.
If the measured value is higher than the threshold, the buzzer turns on as an alert.
If the value is below the threshold, the buzzer remains off.

•• Output
- Gas value displayed on Serial Monitor
- Buzzer alert when gas is detected
