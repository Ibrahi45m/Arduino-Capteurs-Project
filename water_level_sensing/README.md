• Water Level Sensing System

•• Description
This Arduino project detects the water level using a water level sensor and sends the measured value to the Serial Monitor.

•• Components
- Arduino Uno
- Water Level Sensor
- Jumper wires
- LED (optional)

•• How It Works
The water level sensor reads the amount of water.
The Arduino receives the value and displays it on the Serial Monitor.
An LED can be turned on when the water level exceeds a specific threshold.

•• Output
- Water level value displayed on Serial Monitor
