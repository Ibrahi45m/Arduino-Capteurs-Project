• DHT11 Sensor Project

•• Description
This Arduino project uses the DHT11 sensor to read humidity values and display them on the Serial Monitor.

•• Components
- Arduino Uno
- DHT11 Sensor
- Jumper wires

•• How It Works
The DHT11 sensor reads humidity data.
The Arduino receives the value and prints it on the Serial Monitor every few seconds.

•• Output
- Humidity percentage displayed on Serial Monitor
